## Project Description
The AI Chat Application is a web-based project that allows users to interact with an Artificial Intelligence model in real time.  
It is built using HTML, CSS, and JavaScript and integrates the Gemini API to generate intelligent responses.

This application provides a smooth and user-friendly chat interface with support for formatted text and code highlighting.

## Objectives
- To build an interactive AI chatbot
- To integrate external AI API (Gemini)
- To design a responsive and modern UI
- To demonstrate real-time communication with AI

##  Features
-  Real-time AI chat
-  Gemini API integration
-  Modern Glassmorphism UI
-  Search grounding toggle
-  Markdown support (formatted responses)
-  Code syntax highlighting
-  Responsive design

## Technologies Used
- HTML5
- CSS3
- JavaScript 
- Gemini API
- Marked.js (Markdown parsing)
- Highlight.js (Code highlighting)

##Future Improvements
- User authentication (Login/Signup)
- Chat history storage
- Voice input feature
- Dark/Light mode toggle
- Better error handling
